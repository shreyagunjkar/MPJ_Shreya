# Java College Assignments

This repository contains my Java lab assignments and experiments completed as part of my coursework.

## Some Topics Covered

* Object-Oriented Programming (OOP)
* Constructor & Method Overloading
* File Handling
* Exception Handling
* Basic Data Structures

## Repository Structure

* Each file represents a separate experiment or lab assignment.
* Files are named using a consistent pattern like:

  * Lab1.java
  * Lab2.java
  * Lab3a.java
  * Lab4.java
* New experiments are added regularly.

## How to Run

1. Compile the program:
   javac FileName.java

2. Run the program:
   java FileName

##  Tools Used

* Java (JDK)
* IntelliJ
